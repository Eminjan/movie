## movie
* 为了复习前段时间学习的，搭建了一个简单的为电影网站🎏
* 具有的功能用户评论登陆注册，评论和收藏，电影按星级和上映时间分类
* 前端基于Bootstrap3,后端基于Django2.0，为了减缓服务器压力，电影和一些图片数据上传到七牛云播放时从七牛云加载出来
* 由于服务器资源有限，此项目没有上线，而且有些功能还没完全实现，完善中
